# SDG Interview Preparation

## Topics of Focus

**Static classes and members** — Study these thoroughly before the interview.

See [SOLID Principles](../02-concepts/solid.md) and [OOP](../02-concepts/oop.md) for foundational concepts.

## Technical Topics

### C# / .NET
- Static keyword and static members
- Static constructors
- Static classes

### Async/Await
- Task-based asynchronous patterns
- Error handling in async code

### LINQ
- Query syntax
- Method syntax
- Deferred execution

---

## Interview Tips

1. **Be thorough on static** — This is explicitly what they want to test
2. **Use concrete examples** — Show how you'd apply static members in real code
3. **Discuss trade-offs** — Shared state implications, thread safety
