# Rayus / Mahasti Interview Preparation

## Interview Style

**Textbook C# / .NET and SQL questions, backed up with real-world examples from prior work.**

**SQL/Database portion is drilled hardest:**
- Access layer architecture
- Triggers
- Queries
- Stored procedures

---

## Technical Requirements

They're looking for experience with:

### .NET Stack
- .NET Framework (C#, ASP.NET, MVC)
- .NET Core
- API development
- jQuery
- CSS
- Unit testing

### Databases
- Relational databases
- Data-driven application development
- Complex queries and stored procedures

### Methodology
- Agile development

---

## Interview Focus Areas

### C# / .NET
- Fundamental language features
- Async/await patterns
- LINQ
- Entity Framework
- API design

### SQL / Database
- Query optimization (execution plans, indexes)
- Stored procedures (performance, security)
- Triggers
- Complex JOINs and aggregations
- Normalization

### Real-World Context
- How you've applied these technologies in production
- Performance challenges you've overcome
- Database design decisions you've made

---

## Preparation Strategy

1. **Review SQL extensively** — They test this heavily
2. **Prepare real examples** from your work
3. **Know execution plans** — How to read and optimize
4. **Understand stored procedures** — Write, test, debug
5. **Be ready to code** — SQL and C# on the fly
